# Запас прочности — парсеры

| Ситуация | Поведение |
|----------|-----------|
| Нет листа за вчера | Сравнение пропускается, отчёт за сегодня |
| audit_only | Только лист audit, без записи в шаблон |
| Ошибка одной ячейки | Лог + продолжение batch |
| Два cron параллельно | Разные lock-файлы / разные книги |
