# Схема шаблонов A / B

## Лист «Инструкция»

Ячейка → правило → источник (API / Импорт / Формула).

## Лист «Импорт»

Внешние счётчики (SMS, АПП и т.д.) — подтягиваются в ячейки шаблона.

## Лист Audit

Флаги по каждому помещению для сверки с ожиданием.

`SPREADSHEET_TITLE` и `YOUR_HOUSE_ID_*` — в env при деплое.
